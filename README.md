# Forecasting-Budgeting-CV-Artha-Konstruksi 2023
Developed an integrated financial modeling and business planning framework for a construction company, supporting financial analysis, risk assessment, and strategic decision-making.
Note : Confidential client (company name and selected data have been modified to protect confidentiality while preserving the analytical methodology and business insights).
Project goal :
Performed financial ratio, variance, and scenario analyses to identify business risks and improvement opportunities.
Identified a gross margin breakeven point of 10.0% and a minimum collection rate threshold of 88.5%, supporting liquidity and profitability improvement initiatives.
